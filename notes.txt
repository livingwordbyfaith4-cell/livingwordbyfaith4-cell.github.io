YOUR FOLDER
=============
    index.html
    notes.txt
    images/
      header-banner.png
      crossing-the-same-sex-stronghold.png
      with-the-heart-you-believe.png

Keep all of this together in one folder. Double-click index.html to
view your site any time. Nothing updates on its own — after you
change something, save it, then refresh index.html in your browser
to see the change (just like refreshing any webpage).


SWAPPING OUT AN IMAGE (banner or a book cover)
=================================================
1. Rename your new picture to the EXACT SAME name as the one you're
   replacing (e.g. a new banner must be named header-banner.png).
2. Drag it into the "images" folder and click "Replace" when asked.
3. Refresh index.html in your browser. Done — no code needed.


FILLING IN AN EMPTY BOOK SLOT (Book 3 through Book 12)
=========================================================
The Books section already has 10 empty, dashed-outline placeholder
boxes ready for future books — you don't need to create new ones.

For each one you want to fill in, open index.html in Notepad and:

  1. Press Ctrl+F and search for the box's label, e.g.:
        Add Book 3

  2. You'll land on a block that looks like this:

        <div class="book-card is-placeholder">
          <div class="book-cover book-cover--empty"><span>Add Book 3</span></div>
          <h3>Book Title</h3>
          <p class="blurb">Short description goes here.</p>
          <div class="order-buttons">
            <a class="btn btn-primary" href="#" ...>Digital Order</a>
            <a class="btn btn-secondary" href="#" ...>Paperback Order</a>
          </div>
        </div>

  3. Delete just the word  is-placeholder  (leave "book-card") — this
     removes the dashed outline.

  4. Replace this line:
        <div class="book-cover book-cover--empty"><span>Add Book 3</span></div>
     with:
        <div class="book-cover book-cover--photo"><img src="images/your-file-name.png" alt="Your book title"></div>
     (Save your cover picture into the "images" folder first, using
     whatever file name you type after images/ above.)

  5. Replace "Book Title" and the description text with your real
     title and blurb.

  6. Replace both # marks inside href="#" with your real Digital
     Order and Paperback Order links.

  7. Save the file, then refresh index.html in your browser.

Repeat for Book 4, Book 5, and so on whenever you're ready.


UPDATE THE "SCHEDULE A CALL" BUTTON
=======================================
Find:  <a class="counsel-cta" href="#" ...>Schedule a Call</a>
Replace # with a Calendly link, phone number (tel:1234567890), or
email (mailto:you@email.com).


PUTTING THIS ON GITHUB PAGES (free hosting)
================================================
1. Go to github.com and create a free account if needed.
2. Click "+" (top right) → "New repository."
3. Name it, e.g. living-word-by-faith. Set it to "Public." Create it.
4. Click "uploading an existing file."
5. Drag in EVERYTHING — index.html, notes.txt, and the whole images
   folder — then click "Commit changes."
6. Go to the repo's Settings tab → Pages (left sidebar).
7. Under "Branch," choose "main" and folder "/ (root)," then Save.
8. Wait about a minute, refresh — your live link appears at the top:
   https://yourusername.github.io/living-word-by-faith/

To update the live site later, repeat step 5 with your edited files —
GitHub will ask to replace the old ones, say yes.
